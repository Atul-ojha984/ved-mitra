# Ved Mitra

Ved Mitra is a Laravel-powered spiritual services platform for verified pandit booking, Vedic kundli generation, Hindu festival discovery, and Hindi devotional e-book reading.

## Brand

- Name: Ved Mitra
- Tagline: Your Divine Spiritual Companion
- Theme: saffron, gold, deep maroon, temple-inspired gradients

## Core Features

- Public homepage with premium spiritual imagery and Gayatri Mantra ambience controls
- Pandit search, booking, payment, and user dashboards
- Kundli generator with validation, loading state, JSON API, saved history, and PDF download for logged-in users
- Spiritual e-book library with Hindi content, meanings, chapter navigation, bookmarks, night mode, search, and print-to-PDF
- Festival and rashi pages
- Admin and pandit dashboards
- Legal, support, sitemap, SEO, OpenGraph, and footer pages

## Local Development

```bash
composer install
php artisan migrate --seed
php artisan serve
```

Open `http://127.0.0.1:8000`.
